📊 Gestión de Proyectos
Panel interno de seguimiento de proyectos para equipos de consultoría. Construido con HTML, CSS y JavaScript puro, sin frameworks ni dependencias de backend.

✨ Funcionalidades

✅ CRUD completo — crear, editar y eliminar proyectos
📋 Tareas por proyecto — agregá, completá y eliminá tareas dentro de cada proyecto
💬 Comentarios — dejá notas con autor y fecha en cada proyecto
🔍 Buscador en tiempo real — buscá por nombre de proyecto o cliente
🎛️ Filtros por estado — Todos, En curso, En riesgo, Completado
📐 Ordenar proyectos — por nombre, fecha de entrega o progreso
⚠️ Alertas de vencimiento — avisos automáticos para proyectos próximos a vencer o vencidos
📊 Gráfico de dona — resumen visual del estado de todos los proyectos
👥 Estadísticas por responsable — carga de trabajo, progreso promedio y tareas pendientes por persona
🗂️ 3 vistas — Lista, Kanban y Calendario mensual
🌙 Modo oscuro — toggle con preferencia guardada
🖨️ Exportar a PDF — reporte completo con un clic
💾 Persistencia local — todos los datos se guardan en localStorage
